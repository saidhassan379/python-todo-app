# Python To-Do List App

This is a simple command-line To-Do List application written in Python.  
It allows you to add, view, complete, and delete tasks, and saves tasks persistently using a JSON file.

## Features

- Add new tasks
- View all tasks with status (Pending or Done)
- Mark tasks as complete
- Delete tasks
- Tasks are saved between sessions in `tasks.json`

## How to Run

1. Make sure you have Python 3 installed.
2. Clone this repository or download the files.
3. Open your terminal or command prompt in the project folder.
4. Run the program with:

```bash
python todo.py
